# E-Waste Management System

## Overview
The **E-Waste Management System** is a web-based application designed to manage, track, and streamline the collection, recycling, and disposal of electronic waste.  
The system connects **users**, **collection centers**, and **administrators** to ensure environmentally responsible handling of e-waste.

This project focuses on **proper workflow management**, **data consistency**, and **role-based access**, making it suitable for real-world backend development practice.

---

## Problem Statement
Improper disposal of electronic waste leads to:
- Environmental pollution
- Health hazards
- Loss of reusable resources

Manual tracking of e-waste collection is inefficient, error-prone, and non-transparent.  
This system digitizes the process to ensure accountability and traceability.

---

## Features

### User Module
- User registration and authentication
- Submit e-waste pickup requests
- Track request status (Pending / Approved / Collected / Recycled)
- View recycling history

### Collection Center Module
- View assigned pickup requests
- Update collection status
- Maintain inventory of collected e-waste

### Admin Module
- Manage users and collection centers
- Approve or reject pickup requests
- Monitor overall e-waste statistics
- Generate reports
